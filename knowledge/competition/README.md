## When to add a competitor profile

When a row in the table above stops being enough — usually because:

- Sales is losing or winning deals against them more than once a quarter.
- They've shipped something that changes how customers compare us.
- We need to brief a new hire on the matchup.

Copy `_competitor-template.md`, name it `<competitor-slug>.md`, and link it from the table.

## Style

Keep table rows short. Details belong in the per-competitor file. Cite customer evidence when claiming "they win at X" or "customers churn from them for Y" — feedback in InsightStack > opinion.
