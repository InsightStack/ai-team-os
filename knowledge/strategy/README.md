## When to add a doc here

- A priority is large or contested enough to need its own page → copy `_priority-template.md`.
- A new quarter starts and priorities shift → add a `{{YYYY-Qn}}-priorities.md` snapshot before editing this README.
- A strategic narrative deserves a standalone writeup (e.g., a market thesis) → add a slug-named file and link it from the doc index.

If it's just a tweak to an existing priority, edit this README in place.

## Style

Strategy rots fast. Date the last meaningful update at the top of every file in this folder. If a file hasn't been touched in six months, the owner should re-read it and either confirm or update.
